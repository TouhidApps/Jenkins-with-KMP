# Jenkins with KMP
Jenkins CI/CD with Kotlin Compose Multiplatform (KMP)

## How to use:
Keep:
Jenkinsfile and
local.properties
files in your projects root directory in git.
Customise files paremeters as you need and push.

Now you can use it with jenkins
